# trabajofinal Sistema de Asistencia (app)
Integrantes: Gómez Aurea
Curso: 4°3
Materias: Proyecto Informático Y Base de Datos
Este trabajo lo dividí en dos carpetas, Backend y Frontend. En Frontend van a estar los códigos de html, css y javascript, también agregé dos tablas de datos de alumnos random como ejemplo para la app, son dos cursos, 4°3 y 4°4. (Estan en la subcarpeta data-ejemplo). Después en Backend esta la subcarpeta db, en donde esta scripts.sql que tiene la estructura de la base de datos, también connection.js en donde estan las lineas para poder conectar la app con la base de datos. Después en la subcarpeta endpoints esta la ruta para poder leer la lista de los alumnos desde la base de datos y para guardar la asistencia. Y por último server.js que sería el archivo principal del backend que inicia el servidor express, configura la coneexión a Mysql y registra las rutas.
